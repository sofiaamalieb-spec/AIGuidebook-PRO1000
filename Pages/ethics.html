Viktige ting å være obs på ved bruk av KI

Selv om KI-verktøy kan være nyttige læringshjelpemidler, er det flere viktige problemstillinger det er lurt å kjenne til for å bruke dem på en trygg og ansvarlig måte.

Unngå brudd på opphavsrett

KI-verktøy kan i noen tilfeller gjengi opphavsrettsbeskyttet materiale. Derfor er det viktig å være oppmerksom på kildebruk og unngå plagiat, inkludert selvplagiat.

Risiko:

  • • KI-modeller er trent på store mengder internettdata, inkludert opphavsrettsbeskyttede verk. Treningsgrunnlaget kan også være partisk, så vær obs på å ikke videreføre disse skjevhetene.
  • • KI-verktøy kan generere tekst eller ideer som ligger tett opptil opphavsrettsbeskyttede kilder.

Tips:

  • ✓ Omskriv KI-generert innhold med dine egne ord og vær kildekritisk.
  • ✓ Oppgi KI-verktøyet og prompten i kildelisten og henvis i løpende tekst ved direkte sitering.
  • ✓ Be KI om tilbakemelding på tekst du har skrevet selv.

Beskytt sensitiv informasjon

Unngå å dele personlig, konfidensiell eller sensitiv informasjon med KI-verktøy.

Hva du ikke bør dele:

Å dele personlig eller sensitiv informasjon med KI-verktøy kan være risikabelt fordi det du skriver kan lagres, gjenbrukes eller eksponeres uten at du har kontroll.

  • ✗ Personlig informasjon: adresser, telefonnumre, ID-numre, økonomiske detaljer. Dette kan føre til identitetstyveri, svindel og alvorlige personvernbrudd.
  • ✗ Konfidensiell data: private forskningsdata, upublisert arbeid og annen konfidensiell data. Å dele dette kan eksponere sensitive prosjekter, bryte avtaler eller gi uønsket tilgang til konfidensiell informasjon.
  • ✗ Andres informasjon: medstudenters data, intervjuutskrifter med identifiserbar informasjon. Å dele dette kan bryte samtykke og personvernlovgivning, og skade dem du ikke har rett til å dele informasjon om.
  • ✗ Innloggingsinformasjon: passord, kontodetaljer eller koder for flerfaktorautentisering. Dette kan brukes til å ta over kontoene dine og få tilgang til beskyttede data.
  • ✗ Sensitive skoledata eller eksamensbesvarelser: eksamensbesvarelser, vurderte arbeider eller konfidensiell tilbakemelding. Å dele dette kan bryte faglige integritetsregler og utsette deg eller andre for disiplinær reaksjon.

Bruk KI-verktøy på en bevisst måte, og vurder nøye hvilken informasjon som er trygg å dele.

KI-hallusinasjoner: Når KI finner på ting

KI-verktøy kan selvsikkert generere falsk informasjon som høres helt overbevisende ut.

Hva er hallusinasjoner?

KI-hallusinasjoner oppstår når modellen genererer informasjon som er ukorrekt, oppdiktet eller ikke eksisterer, men presenterer det som fakta. Dette skjer fordi:

  • • KI forutsier sannsynlige tekstmønstre, ikke faktisk sannhet.
  • • Den har kanskje ikke informasjon om svært nylige hendelser eller spesialiserte emner.
  • • Den kan 'fylle hull' med plausibel, men falsk informasjon.

Hvordan verifisere informasjon

Det er lurt å faktasjekke KI-generert informasjon før du bruker det i arbeidet ditt.

Strategier for verifisering:

1. Be KI om kilder

Be om: "Vennligst oppgi kilder for disse påstandene" eller "Hvilke studier støtter denne informasjonen?"

2. Kryssreferenser med pålitelige kilder

Sjekk fakta mot akademiske databaser, fagfellevurderte tidsskrifter eller pålitelige referansematerialer.

3. Verifiser URL-er og sitater

Hvis KI gir lenker eller sitater, bør du åpne dem og kontrollere at de eksisterer og støtter det som blir påstått.

Skjevhet i KI-treningsdata

KI-modeller kan reflektere og forsterke skjevheter som finnes i treningsdataene deres.

Forstå KI-skjevhet:

KI-modeller lærer fra data laget av mennesker, og disse dataene kan inneholde skjevheter. Det betyr at KI kan:

  • • Reflektere stereotypier om kjønn, rase, alder, nasjonalitet eller andre egenskaper.
  • • Overrepresentere visse perspektiver mens andre underrepresenteres.
  • • Presentere vestlige eller engelskspråklige perspektiver som universelle.

Hva du bør gjøre:

  • ✓ Vær kritisk: Sett spørsmålstegn ved antakelser og perspektiver i KI-svar.
  • ✓ Søk ulike kilder: Ikke stol utelukkende på KI i emner som involverer sosiale spørsmål.
  • ✓ Vurder kontekst: Anerkjenn at KI kanskje ikke representerer alle kulturelle perspektiver.

Husk: KI er et verktøy som kan støtte læringen din, men det bør brukes sammen med kritisk tenkning, ulike perspektiver og verifiserte informasjonskilder.
